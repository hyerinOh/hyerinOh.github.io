<pre> 태그

  • Preformatted Text
  • 입력한 문장 형태 그대로 브라우저에 표현할 수 있다
  • enter, tab, space가 적용되도록 HTML 형태로 유지
  • CSS 속성 중 white-space: pre-wrap을 사용하면 공백을 코드에 있는 그대로 표시해준다. 코드에 줄바꿈이 없어도 자동 줄바꿈이 된다.
