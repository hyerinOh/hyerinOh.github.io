1. 컴퓨팅 사고 - 1) 2진법


  • 우리가 일상적으로 사용하는 컴퓨터는 어떤 원리로 작동될까?

  • 컴퓨터 과학이란 무엇일까?
  • 컴퓨터가 작동하게 하기 위해 어떤 약속들이 필요할까?

1. 컴퓨터 과학

  1. 컴퓨터 과학이란? 문제 해결에 대한 학문

    • 문제 해결 : input을 받아 output을 만들어내는 과정

    • input -> Computer Science -> output

    • 특히 기계들을 사용할 때는 이러한 input과 output을 어떻게 표현할지에 대해 모두가 동의할 약속(표준) 이 필요


      SO => 컴퓨터 과학의 첫 번째 개념은 정보 자체의 표현 방법 이다.


  1. 2진법

    • 10진법 : 우리가 일상에서 0부터 9까지 총 10개의 기호로 표현하는 방법

<br>

- **2진법** : 오직 0과 1로만 표현하는 방법
  • 2진법은 전기를 켜고 끄는 방식으로 작동하는 컴퓨터에게 적합한 방법

  • 컴퓨터에는 굉장히 많은 스위치(트렌지스터)가 있고 on/off 상태를 통해 0과 1을 표현

컴퓨터는 어떻게 2진법으로 글자, 사진, 영상, 소리 등을 모두 저장할 수 있을까?

  1. bit (binary digit)
  • 2진법에서 하나의 자릿수를 표현하는 단위
  • 정보를 저장하고 연산을 수행하기 위해 0과 1 두 가지 값만 가질 수 있는 bit라는 측정 단위를 쓴다.
  • 디지털 데이터를 여러 비트들로 나타냄으로써 0과 1로도 많은 양의 정보를 저장할 수 있다.
  • 또한 저장되어 있는 데이터를 수정하기 위해 비트에 수학적 연산을 수행할 수 있다.


[출처: https://www.edwith.org/boostcourse-cs-050/lecture/118997/]


  1. byte (비트열)
  • 8개의 비트가 모여 만들어진 것
  • bit(0 또는 1) * 8 = 256개의 서로 다른 바이트가 존재할 수 있음
  • bit 한 개는 많은 양의 데이터를 나타내기에 부족
  • 여러 숫자 조합을 나타내기 위해 byte(비트열)을 사용

컴퓨터는 왜 2진법으로 작동할까?

컴퓨터의 논리회로가 아래와 같이 2 가지 상태를 가지고 있는데
- 전기 신호가 있는 상태 : 1
- 전기 신호가 없는 상태 : 0

만약 10진법으로 작동한다면 컴퓨터는 10가지 상태를 모두 작동시키려 많은 일을 해야한다.

2진법으로 작동하는 것이 컴퓨터가 작동하기 더 쉽게 만들어주고 적은 공간을 차지한다.
