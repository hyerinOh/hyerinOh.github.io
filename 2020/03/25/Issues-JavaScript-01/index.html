1.hasOwnProperty

Mar 25, 2020 in Issues, JavaScript

배열에서 같은 속성끼리 묶기

문제 상황 !
filter된 배열에서 같은 속성끼리 묶기 위해서 공통 key를 객체에 만들고 value를 같은 속성을 넣은 배열로 만들고 싶음

  1. Object.hasOwnProperty(객체.공통으로묶고싶은key)로 해당 key가 있는지 확인
  2. 있으면 obj[객체.공통으로묶고싶은key] = obj[객체.공통으로묶고싶은key].concat(객체)
  3. 없으면 arr(새로 변수) 생성해서 arr.push(객체) 해주고 obj[객체.공통으로묶고싶은key] = arr
