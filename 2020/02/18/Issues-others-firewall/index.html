방화벽 또는 외부로의 차단?

문제 상황 !
github에 push하려 했더니 timeout fatal 에러 발생

  • 보안 프로그램이 깔려있는 인터넷 환경이다보니 방화벽 문제 또는 외부로의 차단 때문인 것 같음
  1. 터미널에서 git config --local -e를 치고
  2. url 부분을 https://github.com/이름/repo.git으로 변경하고 (insert, delete키 이용)
  3. Vim 명령어인 :w로 저장
  4. git push -u origin master가 정상적으로 동작하게 된다.
