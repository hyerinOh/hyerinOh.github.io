사이드 메뉴가 스크롤 안되는 이슈

문제 상황 !
iframe으로 되어있는 사이드 메뉴가 스크롤이 되지 않아서 아래에 있는 컨텐츠들을 못 보여주는 상황

  • height를 100vh로 바꿔주고(사이드 메뉴 높이를 꽉차게 만들고 싶음) menu div를 overflow: scroll로 바꿔줌(가로/세로 모두 스크롤이 필요)

  • vh : view-port height를 의미하는 단위. 사용자가 보는 스크린을 100으로 봐서 1vh는 view-port height의 100분의 1

  • overflow : 요소내의 컨텐츠 양이 너무 많아서 한 번에 보여주기 힘들 때 어떻게 보여줄지를 정함
    • visible: 기본 값. 넘칠 경우 컨텐츠가 요소 밖으로 보여짐
    • hidden: 넘치는 부분이 잘려서 보여지지 않음
    • scroll: 가로/세로 모두 스크롤바가 추가됨
    • auto: 컨텐츠 양에 따라 스크롤바를 추가할지 자동으로 결정
