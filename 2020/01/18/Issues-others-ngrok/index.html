ngrok

문제 상황 !
현재 Mac OS를 사용하는 중이라 IE 테스트를 할 수 없어 윈도우 유저에게 테스트를 부탁해야 했는데 같은 공간에 있지 않아도 url을 받아 IE를 테스트할 수 있었음

  • 사용 방법

    1. ngrok 홈페이지에서 다운받은 후 압출 풀기
    2. 터미널에서 해당 경로를 입력후 포트 번호 입력
      ex ) ./Downloads/ngrok http 3000
    3. Forwarding 주소를 상대에게 공유
